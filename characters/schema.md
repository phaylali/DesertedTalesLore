# Characters Schema

This document outlines the file structure and schema for each character within the `characters` directory.

Each character (historical or fictional) will have a file with this structure:

```markdown
# [Character Name]

## Profile
*   **Full Name:**
*   **Birth:**
*   **Death:**
*   **Place of Birth:**
*   **Tribal Origin:**
*   **Faction/Allegiance:**
*   **Role:** (e.g., Protagonist, Blacksmith, Religious Leader)

## Biography
A detailed biography of the character.

## Skills and Abilities
A description of the character's skills.

## Game-Specific Information
*   **Role in the Story:**
*   **Associated Quests:**
```
