# Tribes Schema

This document outlines the file structure and schema for each tribe within the `tribes` directory.

Each tribe will have its own file with the following structure:

```markdown
# [Tribe Name]

## Origins and History
A brief history of the tribe.

## Territory
The geographical region the tribe inhabits.

## Culture and Society
*   **Social Structure:** How the tribe is organized.
*   **Customs and Traditions:** Unique customs and beliefs.
*   **Alliances and Rivalries:** The tribe's relationship with other tribes and factions.

## Military
*   **Warriors:** Description of the tribe's warriors and their fighting style.
```
