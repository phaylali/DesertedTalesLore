# Events Schema

This document outlines the file structure and schema for each event within the `events` directory.

Each event (historical or fictional) will have a file with this structure:

```markdown
# [Event Name]

## Overview
*   **Date:**
*   **Location:**
*   **Factions Involved:**
*   **Historical Basis:** (Real or fictional)

## Description
A detailed description of the event.

## Game-Specific Information
*   **Player's Role:** How the player is involved.
*   **Possible Outcomes:**
```
